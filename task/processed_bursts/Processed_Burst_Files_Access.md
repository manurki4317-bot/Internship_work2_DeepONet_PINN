# Access to Generated Burst Processing Files

During the processing of the ABF recordings, a large number of
intermediate and output files are generated (processed datasets,
extracted bursts, figures, auxiliary files, etc.).

Because including all of these files directly in the repository is
impractical, they have been uploaded to a shared Google Drive folder.

You can access and download all the required files using the following
shared link:

**Google Drive folder:**
https://drive.google.com/drive/folders/1fWLYMaFt2eRL4_8Qx31UlD3oLeZQmr9L?usp=sharing

This folder contains the files needed to work with the processed bursts
and reproduce the corresponding analyses.

If additional files are generated in the future, they may also be added
to this shared folder so that all resources remain available from a
single location.
